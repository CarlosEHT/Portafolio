Experiencia

Sitio web biográfico para el Dr. José Carlos García Ramírez

Desarrollo de un sitio web bibliográfico y de difusión profesional para el Dr. José Carlos García Ramírez (https://josecarlosgarcia.org/), con el objetivo de presentar su trayectoria académica y producción intelectual. La plataforma permite administrar y consultar libros publicados —que cuentan con un sistema de comentarios tipo foro—, además de artículos indexados, cursos impartidos, columnas periodísticas y contenidos multimedia como galería de fotos y videos de YouTube. Incluye una agenda de eventos futuros, autenticación con roles diferenciados (usuario/administrador), y envío automático de correos electrónicos durante el registro, inicio de sesión y otras interacciones clave.
Los archivos del sitio se gestionan mediante un bucket de Google Cloud Storage para un manejo eficiente y escalable de recursos. El sitio está indexado en Google Search Console y en Bing Webmaster Tools, lo que permite una mayor visibilidad y control sobre su presencia en buscadores.



HTML CSS JavaScript Bootstrap Full Calendar Ajax jQuery Python Django PostgreSQL Render Gmail Google Cloud Storage Google Search Console Bing Webmaster Tools GitHub

Sistema de Gestión para Cibercafé y Papelería

Desarrollé una aplicación de escritorio en Java para la gestión integral de un cibercafé, enfocada en automatizar procesos operativos y administrativos. El sistema incluye autenticación con múltiples roles (gerente y empleado), cada uno con privilegios diferenciados. Permite realizar operaciones tipo CRUD en las siguientes áreas:

  • Gestión de empleados y sus horarios laborales.
  • Gestión de cierres diarios, donde se calculan las ganancias generadas por:
    • Renta de equipos de cómputo o consolas de videojuegos (con cobro basado en tiempo).
    • Venta de artículos de papelería.
  • Control de tipos de nómina.
  • Gestión de renta de equipos y registro de sesiones activas.
  • Inventario y ventas de productos de papelería.
  • Administración de productos disponibles para la venta.
El sistema permite mantener un control centralizado y eficiente de las operaciones diarias del negocio, facilitando la toma de decisiones y la supervisión del personal.
